# Умная игрушка для кота (ESP32 + FreeRTOS)

Этот проект создан для Arduino IDE и содержит:
- скетч `smart_cat_toy.ino`;
- управление двумя SG90 (360 + 180);
- датчик расстояния VL53L0X;
- OLED SSD1306 по SPI;
- управление лазером через транзистор;
- простую звуковую индикацию.

## 1) Схема подключений (актуальная)

### OLED SSD1306 (SPI) — как у вас уже подключено
- `OLED_MOSI = GPIO23`
- `OLED_CLK  = GPIO18`
- `OLED_DC   = GPIO2`
- `OLED_CS   = GPIO5`
- `OLED_RESET= GPIO4`

### Остальные пины
- `SERVO_PAN_360  -> GPIO19`
- `SERVO_TILT_180 -> GPIO27`
- `LASER_CTRL     -> GPIO26` (через транзисторный ключ)
- `SPEAKER        -> GPIO25` (PWM/LEDC)
- `VL53L0X SDA    -> GPIO21`
- `VL53L0X SCL    -> GPIO22`

## 2) Питание и безопасность

- Сервы питать от отдельной линии 5V (не от 3.3V ESP32).
- Все земли объединить: `GND ESP32 = GND серв = GND датчика = GND лазера`.
- Лазер не подключать напрямую к GPIO: только через транзистор.
- Поставить электролит 470–1000 мкФ на 5V рядом с сервами.

## 3) Библиотеки в Arduino IDE

Убедитесь, что установлены:
- `Adafruit SSD1306`
- `Adafruit GFX`
- `Adafruit VL53L0X`
- `ESP32Servo`

## 4) Как запустить

1. Откройте в Arduino IDE файл `smart_cat_toy.ino`.
2. Выберите плату ESP32 (вашу модель dev-платы).
3. Выберите COM-порт.
4. Нажмите **Verify/Compile**.
5. Нажмите **Upload**.
6. Откройте **Serial Monitor** (115200 бод).

## 5) Порядок отладки для новичка

1. **Только ESP32 + OLED**  
   Проверить, что экран инициализируется (в Serial будет сообщение, на экране — строки статуса).

2. **Добавить VL53L0X**  
   В Serial должно быть `VL53L0X OK`. На экране — адекватная дистанция.

3. **Подключить серво (без лазера)**  
   Проверить, что 180° плавно ходит, а 360° старт/стоп работает без дерганий.

4. **Добавить лазер через ключ**  
   Проверить включение/выключение по режимам.

5. **Подключить динамик**  
   Проверить звуки в режимах `ACTIVE PLAY` и `CALM PLAY`.

## 6) Типовые проблемы и решения

- **Сервы дергаются / ESP32 перезагружается**  
  Обычно причина в питании. Усилить 5V, добавить конденсатор, укоротить провода питания.

- **VL53L0X не найден**  
  Проверить SDA/SCL, питание 3.3V, общий GND.

- **OLED не стартует**  
  Проверить SPI-пины и наличие библиотеки `Adafruit SSD1306`.

- **Лазер не выключается/не включается**  
  Проверить схему транзисторного ключа и общий GND.

## 7) Что можно улучшить дальше

- Добавить калибровку серв через Serial-команды.
- Сделать несколько игровых сценариев и случайное переключение.
- Ввести "тихий ночной режим".
- Добавить статистику: время активности, количество срабатываний датчика.
